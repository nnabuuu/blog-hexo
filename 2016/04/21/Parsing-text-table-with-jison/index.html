什么是Jison

Jison的官网介绍是：

Jison是一个”Bison in JavaScript”。也就是说，首先Jison是一个类似Bison的parser生成器。区别在于，Jison使用JavaScript编写，可以同时用于NodeJS和浏览器执行环境中。

Jison的安装请参照http://zaa.ch/jison/docs/，在NodeJS的开发环境中使用`npm install jison -g`即可完成安装。
